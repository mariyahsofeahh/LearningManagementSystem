Course Description

This course provides a deep dive into the Object-Oriented paradigm. Students will learn how to design complex software systems by modeling real-world entities into software objects. The course emphasizes clean code practices and design patterns essential for modern software engineering.

Weekly Syllabus

Week 1-2: Introduction to Java & OOP Basics

Setting up the environment, understanding classes, objects, and constructors.

Week 3-5: The Four Pillars of OOP

Deep dive into Encapsulation, Inheritance, Polymorphism, and Abstraction with practical labs.

Week 6-8: Exception Handling & File I/O

Learning how to build robust applications that handle errors gracefully and manage persistent data.
